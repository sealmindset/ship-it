## Before going live — prerequisites

- [ ] **App identity / SSO registration** — does this app need users to log in?
- [ ] **SSL/TLS certificate** — does this app have a web address that needs to be secure?
- [ ] **Access control / permissions** — who can access this in production?
- [ ] **DNS / web address setup** — does this need a URL people can visit?
- [ ] **Network / firewall rules** — does this need to talk to internal systems?

> Items left unchecked will need input from your DevOps team.
> /ship-it flagged these so nothing gets missed.
